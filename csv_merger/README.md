# CSV Merger

This tool is designed to merge multiple CSV files into a single CSV file. It takes input from a directory containing the CSV files, processes them, and outputs a consolidated CSV file.

## Features
- Merges multiple CSV files.
- Customizable output filename.
- Error handling for incompatible formats.

## Usage
Simply run the tool with the directory of CSV files as the input parameter.

## Requirements
- Python 3.x
- Pandas library (install using `pip install pandas`)
